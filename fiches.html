Remplissage automatique par IAJSON officiel V4
Étapes : 1. Exporte le modèle JSON. 2. Donne-le à une IA avec la marque et la référence. 3. Demande-lui de retourner uniquement le JSON rempli. 4. Importe le fichier ou colle directement le JSON ci-dessous.
L’import accepte aussi un JSON entouré de ```json, un objet direct, un objet product, une liste products, ainsi que plusieurs noms de champs en français.
Dossier officiel autonome1 ZIP • 1 JSON • aucun fichier externe
Import autonome : le ZIP peut contenir uniquement produit.json. Le logo, le vrai QR EPREL, l’étiquette énergie officielle et les PDF sont encodés dans ce JSON puis reconstruits automatiquement.
Le ZIP autonome ne dépend d’aucun fichier placé à côté. Les images et PDF sont stockés en Base64 dans produit.json, donc le ZIP peut être déplacé et importé sur n’importe quel ordinateur.
